How to Use the Layout

This layout allows you to:

  • Select a floor from the left sidebar to set as the background.
  • Drag and drop assets onto the canvas or click to place them in the tech/hardware areas.
  • Click on assets to select and edit them (scale, flip, z-index adjust, delete) even when zoomed in/out.
  • Use the zoom buttons to adjust the view (cannot zoom out below the original size).
  • Click the maximize button to view the canvas in full-screen mode.
